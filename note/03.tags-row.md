Tags Row
Purpose
Create a horizontal row of colored tags with ability to add, remove, and manage tags. Tags use soft pastel colors from a predefined palette.

The Prompt
You are a senior frontend developer creating a tags row component for a PKM app called Memry.

## TASK
Create a horizontal tags row with colored tag chips and an add button.

## PROPS INTERFACE
```typescript
interface TagsRowProps {
  tags: Tag[];                              // Current tags on note
  availableTags: Tag[];                     // All tags in user's library
  recentTags: Tag[];                        // Recently used tags
  onAddTag: (tagId: string) => void;        // Add existing tag
  onCreateTag: (name: string, color: string) => void;  // Create new tag
  onRemoveTag: (tagId: string) => void;     // Remove tag from note
  disabled?: boolean;                       // Disable editing
}

interface Tag {
  id: string;
  name: string;
  color: string;      // Color name from palette (e.g., "rose", "purple")
}
```

## OUTPUT

Create a TagsRow component that:
1. Displays tags as colored pill chips in horizontal layout
2. Shows remove button on tag hover
3. Opens tag input popup on add button click
4. Supports searching/filtering existing tags
5. Allows creating new tags with color picker
6. Wraps tags to multiple lines if needed
7. Handles empty state gracefully
8. Is fully accessible
9. Uses the specified color palette

Implementation Notes
Key Techniques Used

Color Mapping

Create a tagColors object mapping color names to bg/text values
getTagColors(colorName) returns { background, text }


Popup Positioning

Use position: absolute relative to add button
Consider viewport edges (flip up if near bottom)
Use library like floating-ui for robust positioning


Search/Filter

Filter availableTags by typed text (case-insensitive)
Show "Create [name]" if no exact match


Tag Storage

Tags stored globally in user's library
Notes reference tags by ID
Color is property of tag, not per-note
